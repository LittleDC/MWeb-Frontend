集合应用类型

Object

创建

  1. new操作符 new Object()
  2. 对象字面量（object literal）

获取

  1. 点语法
  2. 中括号 --- 可以通过变量访问属性

Array

创建数组

  1. Array.from()
    第一个参数是一个类/伪数组对象
    第二个参数是映射函数参数（可选）

  2. Array.of()
    可以把一组参数转换为数组

Array.of(1,2,3,4) // [1,2,3,4]

检测数组 value

  1. value instanceof Array（假定只有一个全局执行上下文）
  2. Array.isArray(value)

迭代器方法

let arr = ['foo', 'bar']

  1. arr.keys() //[0, 1]
  2. arr.values() //['foo', 'bar']
  3. arr.entries() //[[0, 'foo'], [1, 'bar']]

复制和填充方法

  1. arr.fill() 批量复制
  2. arr.copyWinthin() 填充

转换方法

  1. arr.toLocaleString()
  2. arr.toString() 每个值调用toString()方法后','拼接 返回字符串
  3. arr.valueOf() 返回数组本身
  4. arr.join(arg) 拼接字符串 arg分隔符

栈方法 队列方法

  1. 栈：arr.push() / arr.pop()
  2. 队列：
    1. arr.push() / arr.shift()
    2. arr.pop() / arr.unshift()

排序方法 --- 会改变原数组arr

  1. arr.reverse() 将数组反向
  2. arr.sort() 默认按照字符串升序重新排列数组元素

操作方法

  1. arr.concat() 不改变原数组
  2. arr.slice() 不改变原数组
  3. arr.splice(起始位置，删除个数，插入元素...) 改变原数组 删除，插入，替换

搜索和位置方法

  1. 严格相等搜索 arr.indexOf() / arr.lastIndexOf() / arr.includes() es7新增
  2. 断言函数搜索 arr.find() / arr.findIndex()

迭代方法 --- 都不改变原数组arr

对数组每一项都运行传入的函数

方法 返回值
arr.filter() 函数返回true的项会组成数组之后返回
arr.forEach() 没有返回值，本质上相当于for循环遍历数组
arr.map() 返回由每次函数调用的结果构成的数组
arr.every() 如果对每一项函数都返回true，则该方法返回true
arr.some() 如果有一项函数返回true，则该方法返回true

归并方法

  1. arr.reduce()
    [1,2,3,4,5].reduce((prev, cur, index, array) => prev + cur) //15
  2. arr.reduceRight()

Tips

  1. 使用length属性可以方便的向数组末尾添加元素（length会自加一）
    arr[arr.length] = 'blue'
  2. 数组最多可以包含 2^32-1 个元素

定型数组？

Map 键/值存储机制

es6新增
Map可以使用任何JavaScript数据类型作为键，其内部使用SameValueZero比较操作。

const m = new Map()

const a = 0/"", b = 0/"", // NaN
      pz = +0, nz = -0

console.log(a === b) //false
console.log(pz === nz) //true

m.set(a, "foo")
m.set(pz, "bar")

alert(m.get(b)) //foo
alert(m.get(nz)) //bar

基本API

  1. 创建 const m = new Map()
  2. m.set() 方法添加键/值对，返回映射实例，因此可以连缀使用
  3. m.get() / m.has() 方法进行查询
  4. m.delete() / m.clear() 方法删除值
  5. m.size属性获取映射中键值对的数量

顺序与迭代

  1. m.entries() 默认迭代器 -> 可以直接对映射实例使用扩展操作
  2. m.forEach() 回调方式
  3. m.keys() / m.values() 分别返回以插入顺序生成键和值的迭代器

键和值在迭代器遍历时是可以修改的，但映射内部的引用则无法修改

选择Object还是Map

  1. 内存占用 --- 给定固定大小的内存，Map大约可以比Object多存储50%的键值对
  2. 插入性能 --- 涉及大量插入操作选Map的性能更佳
  3. 查找速度 --- 涉及大量查找操作选Object更好一些（浏览器引擎可以进行优化）
  4. 删除性能 --- 涉及大量删除操作选Map的性能更佳（Map的delete()操作都比插入和查找更快）

Set 集合数据结构

es6新增
类似加强的Map，可以包含任何JavaScript数据类型作为值，集合也使用SameValueZero操作。
键值相等？

基本API

  1. 创建 const s = new Set()
  2. s.add() 方法增加值，返回集合实例，故可连缀使用
  3. s.has() 方法查询
  4. s.delete() / s.clear() 方法删除元素
  5. s.size 属性获取元素数量

顺序与迭代

  1. s.values() / s.keys() 默认迭代器，后者为别名
    s.values === s[Symbol.iterator] //true
    s.keys === s[Symbol.iterator] //true
  2. s.forEach() 回调方式
