JS整除，取余

1.取整

// 丢弃小数部分,保留整数部分
parseInt(7/2)　　// 3

// 向上取整,有小数就整数部分加1
Math.ceil(7/2)　　// 4

// 向下取整,丢弃小数部分
Math.floor(7/2)　　// 3

// 四舍五入
Math.round(7/2)　　// 3

2.取余

// 1. 取余
7%2　　// 1
