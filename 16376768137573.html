RFC Request For Comments

性能指标

速率 带宽 吞吐量

  • 速率：
    比特 1/0 位
    b/s kb/s Mb/s Gb/s

1Byte(字节) = 8bit(比特)
1KB = 2^10B = 1024B = 1024 * 8b

  • 带宽：网络设备所支持的最高速度（理想发送）

  • 吞吐量：吞吐量受网络的宽带或网络的额定速率的限制（接收）

时延 时延带宽积 往返时延RTT 利用率

  1. 时延：
    发送时延 = 数据长度/信道带宽（发送速率）
    传播时延 取决于电磁波传播速度和链路长度 = 信道长度/电磁波信道传播速率
    排队时延
    处理时延

  2. 时延带宽积：容量（bit为单位）
    时延带宽积（bit） = 传播时延（s） * 发送带宽（b/s）

  3. 往返时延RTT：
    RTT = 传播时延 * 2 + 末端处理时间

  4. 利用率：
    信道利用率 网络利用率

分层结构

SDU(服务数据单元) + PCI(协议控制信息) = PDU(协议数据单元)

  1. 7层OSI参考模型：法定标准
    物理层 -> 数据链路层 -> 网络层 -> 传输层 -> 会话层 -> 表示层 -> 应用层
    通信子网(数据通信) | 传输层 | 资源子网(数据处理)

传输层功能：
可靠传输/不可靠传输，差错控制，流量控制，复用分用

  1. 4层TCP/IP参考模型：事实标准
