计算机网络

第一章 概要

标准化工作

RFC Request For Comments

性能指标

速率 带宽 吞吐量

  • 速率：
    比特 1/0 位
    b/s kb/s Mb/s Gb/s

1Byte(字节) = 8bit(比特)
1KB = 2^10B = 1024B = 1024 * 8b

  • 带宽：网络设备所支持的最高速度（理想发送）

  • 吞吐量：吞吐量受网络的宽带或网络的额定速率的限制（接收）

时延 时延带宽积 往返时延RTT 利用率

  1. 时延：
    发送时延 = 数据长度/信道带宽（发送速率）
    传播时延 取决于电磁波传播速度和链路长度 = 信道长度/电磁波信道传播速率
    排队时延
    处理时延

  2. 时延带宽积：容量（bit为单位）
    时延带宽积（bit） = 传播时延（s） * 发送带宽（b/s）

  3. 往返时延RTT：
    RTT = 传播时延 * 2 + 末端处理时间

  4. 利用率：
    信道利用率 网络利用率

分层结构

SDU(服务数据单元) + PCI(协议控制信息) = PDU(协议数据单元)

  1. 7层OSI参考模型：法定标准
    物理层 -> 数据链路层 -> 网络层 -> 传输层 -> 会话层 -> 表示层 -> 应用层
    通信子网(数据通信) | 传输层 | 资源子网(数据处理)

传输层功能：
可靠传输/不可靠传输，差错控制，流量控制，复用分用

网络层（IP层）：
把分组从源端传到目的端，为分组交换网上的不同主机提供通信服务。
传输单位：数据包
功能：路由选择，流量控制，差错控制，拥塞控制

数据链路层功能：
成帧，差错控制，流量控制，访问控制

  1. 4层TCP/IP参考模型：事实标准
    网络接口层 -> 网际层 -> 传输层 -> 应用层

  2. 5层体系结构
    物理层 -> 数据链路层 -> 网络层 -> 传输层 -> 应用层

tip

应用层 传输层 网络层 数据链路层 物理层
传输单元名词 报文 报文段 IP数据报 => 分组 帧 比特流

第二章 物理层

主要任务：确定于传输媒体接口有关的一些特性。

码元：用一个固定时长的信号波形（数字脉冲）代表不同离散数值的基本波形。

失真的一种现象 --- 码间串扰 ：高频信号无法通过

编码与调制

传输介质：数据传输系统中在发送设备和接收设备之间的物理通路。 != 物理层

物理层设备：

  1. 中继器：对信号进行再生和还原
  2. 集线器(多口中继器)：对信号进行再生放大转发，星形拓扑（广播）
