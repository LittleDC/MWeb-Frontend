eslint - vue

前端项目技巧
vscode配置eslint自动格式化代码

  • 重点：vscode，其余软件可能无效，，

以下只代表基本步骤，可能会有些出入，建议多百度尝试尝试

参考：https://github.com/vuejs/eslint-config-vue

1.vscode插件安装：

  1. Vutur：vue文件的高亮，提示等作用
  2. ESLint：代码格式化 主要针对js格式
  3. prettier：代码格式化 主要针对html格式

2.确保项目有eslint

npm install --save-dev eslint

4.vscode setting

配置一下vscode的配置文件.vscode/settings.json
实现ctrl + s保存文件时自动格式化代码

// .vscode/settings.json
{
    "eslint.validate": ["javascript", "javascriptreact", "vue"],
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    },
    "vetur.format.defaultFormatter.html": "prettyhtml",
    "vetur.format.defaultFormatter.js": "none", // 禁用vetur的JS格式化，交给eslint处理
    "vetur.format.defaultFormatterOptions": {
      "js-beautify-html": {
        "wrap_attributes": "auto"
      },
      "prettyhtml": {
        "printWidth": 150,
        "singleQuote": false,
        "wrapAttributes": false,
        "sortAttributes": false,
      }
    },
}
