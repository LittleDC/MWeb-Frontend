对象,类与面对对象编程

理解对象

属性的类型(描述符)

使用一些内部特性来描述属性的特征。表示：[[Enumerable]]

  1. 数据属性(数据描述符)
  2. 访问器属性(存取描述符)

一个描述符只能是以上两者其中之一；不能同时是两者。

描述符可拥有的键值：

configurable enumerable value writable get set
数据描述符 ✓ ✓ ✓ ✓ ✕ ✕
存取描述符 ✓ ✓ ✕ ✕ ✓ ✓

  1. configurable
    当且仅当该属性的 configurable 键值为 true 时，该属性的描述符才能够被改变，同时该属性也能从对应的对象上被删除。
    一个属性被定义为不可配置之后，就不能再变回可配置了。再次调用Object.defineProperty()并修改任何非 writable 属性会导致错误

  2. enumerable
    当且仅当该属性的 enumerable 键值为 true 时，该属性才会出现在对象的枚举属性中。

  3. value
    该属性对应的值。可以是任何有效的 JavaScript 值（数值，对象，函数等）。

  4. writable
    当且仅当该属性的 writable 键值为 true 时，属性的值，也就是上面的 value，才能被赋值运算符 改变。

  5. get
    当访问该属性时，会调用此函数。执行时不传入任何参数，但是会传入 this 对象（由于继承关系，这里的this并不一定是定义该属性的对象）。该函数的返回值会被用作属性的值。

  6. set
    当属性值被修改时，会调用此函数。该方法接受一个参数（也就是被赋予的新值），会传入赋值时的 this 对象。

  • 默认情况下，所有直接定义在对象上的 configurable、enumerable 和 writable 属性都是true
  • 访问器属性是不能直接定义的，必须使用Object.defineProperty().

定义描述符

Object.defineProperty()

Object.defineProperty(obj, prop, descriptor)

方法不指定属性时的默认值：

  • 拥有布尔值的键 configurable、enumerable 和 writable 的默认值都是 false。
  • 属性值和函数的键 value、get 和 set 字段的默认值为 undefined。

Object.defineProperty

"use strict";
let book = {
	year_: 2017,
	edition: 1
}
Object.defineProperty(book, 'year', {
	get(){
		return this.year_
	},

	set(newValue) {
		if (newValue > 2017) {
			this.year_ = newValue
			console.log(this.year, this.year_)
			this.edition += newValue - 2017
		}
	}
})

book.year = 2018
console.log(book.edition) //2

Object.defineProperties()

Object.defineProperties(obj, props)

"use strict";
let book = {}
Object.defineProperties(book, {
	year_: {
	   // writable: false,
		value: 2017
	},

	edition: {
	   // writable: false,
		value: 1
	},

	year: {
		get(){
			return this.year_
		},

		set(newValue) {
			if (newValue > 2017) {
				this.year_ = newValue
				console.log(this.year, this.year_)
				this.edition += newValue - 2017
			}
		}
	}
})

book.year = 2018 // error!

读取描述符

方法：Object.getOwnPropertyDescriptor(obj, prop)
返回一个对象，包含各自对应的属性。

方法：Object.getOwnPropertyDescriptors(obj)

let book = {}
Object.defineProperties(book, {
	year_: {
		value: 2017
	},
	edition: {
		value: 1
	},
	year: {
		get(){
			return this.year_
		},
		set(newValue) {
			if (newValue > 2017) {
				this.year_ = newValue
				console.log(this.year, this.year_)
				this.edition += newValue - 2017
			}
		}
	}
})

console.log(Object.getOwnPropertyDescriptors(book))

[Log] Object
edition: {value: 1, writable: false, enumerable: false, configurable: false}
year: {get: function, set: function, enumerable: false, configurable: false}
year_: {value: 2017, writable: false, enumerable: false, configurable: false}
