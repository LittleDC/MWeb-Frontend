some tips

clash for windows

  1. install the TAP Device
  2. config the Mixin:
mixin:
  dns:
    enable: true
    enhanced-mode: redir-host
    listen: 0.0.0.0:53
    nameserver:
      - https://dns.alidns.com/dns-query
      - https://doh.pub/dns-query
      - tls://dns.rubyfish.cn:853
    fallback:
      - tls://1.0.0.1:853
      - tls://dns.google:853
  1. open the Mixin & the network will reconnect

rwts-pdfwriter for mac

install:
brew install --cask rwts-pdfwriter

uninstall:
brew uninstall rwts-pdfwriter -f
